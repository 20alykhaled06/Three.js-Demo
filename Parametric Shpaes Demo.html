Inputs:

  • Mouse (left click): Look around
  • Mouse (scroll): Zoom in/out
  • N, L, P: Switch between shading types
  • W: Wireframe
  • Arrow keys: Rotate object
  • R: Toggle rotation
  • [: Zoom in
  • ]: Zoom out
  • C: Toggle continuous LOD
  • A: Animate between 2 parametric shapes
  • F/S: Animation speed up/down
  • T: Toggle animation pause
  • Z: Animate between 25 parametric shapes
  • J/K: Show next/prev shape
  • X+num+X: Show shape number X (X in [1,25])
